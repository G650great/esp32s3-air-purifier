# OLED UI Interface Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 根据《项目手册》的要求，为 ESP32-S3 智能空气净化器构建并完善 0.96 寸 OLED (SSD1306) 的 UI 界面，包括双字体支持、启动画面和主控数据面板。

**Architecture:** 
UI 界面分为底层驱动层和显示业务层。底层驱动层 (`oled.c`) 负责 I2C 通信、SSD1306 初始化、基本图形绘制及两种字体（6x8 和 8x16）的渲染；显示业务层 (`display.c`) 负责调度底层接口，绘制符合项目手册要求的启动画面 (Splash Screen) 及包含温度、湿度、风扇、空气质量、WiFi 状态的主数据面板 (Main Dashboard)。

**Tech Stack:** C 语言, ESP-IDF (I2C Driver), SSD1306 驱动指令集

- [ ] **Step 1: 编写测试用例验证当前大字体功能**

修改 `main/esp32s3/oled_test.c` 的 `oled_test_task` 函数，添加大字体显示测试：
```c
#include "oled.h"
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"
#include "esp_log.h"

static const char *TAG = "OLED_TEST";

void oled_test_task(void *arg) {
    ESP_LOGI(TAG, "OLED test starting...");
    oled_init();
    oled_clear();
    
    oled_set_font_small();
    oled_draw_string(0, 0, "Small Font");
    
    oled_set_font_large();
    oled_draw_string(0, 20, "Large Font");
    
    oled_update();
    ESP_LOGI(TAG, "OLED test done");
    while (1) { vTaskDelay(pdMS_TO_TICKS(1000)); }
}
```

// 修改 oled_draw_char 以支持 current_font_size
void oled_draw_char(int x, int y, char c) {
    if (c < 32 || c > 127) c = '?';
    if (current_font_size == 0) { // Small font (6x8)
        const uint8_t *glyph = font6x8[(uint8_t)c - 32];
        for (int col = 0; col < 6; col++) {
            uint8_t bits = glyph[col];
            for (int row = 0; row < 8; row++) {
                if (bits & (1 << row)) {
                    oled_draw_pixel(x + col, y + row);
                }
            }
        }
    } else { // Large font (8x16)
        const uint8_t *glyph = font8x16[(uint8_t)c - 32];
        for (int col = 0; col < 8; col++) {
            uint8_t bits1 = glyph[col];
            uint8_t bits2 = glyph[col + 8];
            for (int row = 0; row < 8; row++) {
                if (bits1 & (1 << row)) oled_draw_pixel(x + col, y + row);
                if (bits2 & (1 << row)) oled_draw_pixel(x + col, y + row + 8);
            }
        }
    }
}
